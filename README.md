# tacash36

![](img/img00001.jpg)

## What is tacash36?

tacash36 is small keyboard.
<br>
tacash36 has 36keys.
<br>

This layout has this image.
<br>

![](img/img00002.jpg)

This keyboard use Pro micro or BLE Micro Pro, run qmk_firmware.
<br>
With this keyboard, you can edit your favorite keymap using vial.


<br>
